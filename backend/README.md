# Backend – Pharmacy Academy

Phần server / API / database của hệ thống.

> 🚧 **Chưa phát triển.** Sẽ bổ sung sau khi nhóm chốt công nghệ (Node.js/Express, PHP/Laravel, hoặc Python/Django…).

## Dự kiến chức năng

- Đăng ký / đăng nhập (JWT hoặc session)
- Quản lý tài khoản & phân quyền (admin, giảng viên, sinh viên)
- CRUD khóa học, bài giảng, danh mục thuốc
- API chat realtime (WebSocket / Socket.IO)
- Lưu trữ dữ liệu & ảnh người dùng

# Docs – Tài liệu CNPM

Nơi lưu tài liệu môn Công nghệ Phần mềm.

## Dự kiến

- `SRS.md` – Software Requirements Specification (đặc tả yêu cầu)
- `use-case.md` – Use case chi tiết
- `diagrams/` – Sơ đồ ERD, class diagram, sequence diagram, mockup
- `meeting-notes/` – Biên bản họp nhóm
- `report/` – Báo cáo tổng kết đồ án
